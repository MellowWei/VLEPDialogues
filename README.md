## 这个文件是什么

这是振动—语言—伦理物理学和**当代六个相邻学派**的精确交锋档案。

**不是辩论记录(debate transcripts)。是本体相遇(ontological encounters)。**

学派不需要"赢"这六个对话者。需要的是 ——
- 精确定位双方在哪里能交锋
- 精确定位双方在哪里不能交锋
- 诚实评估各自能给对方什么
- 诚实评估方法论真正不兼容处

主要给四类读者:
1. **依恋理论 / 临床心理学家** —— 看本学派与 Bowlby 的精确关系
2. **爱的哲学家** —— 看本学派与 Badiou / 韩炳哲的兄弟之争
3. **AI / 情感计算工程师** —— 看本学派与 Picard 的层级互补
4. **意识科学家 / 物理学家** —— 看本学派与 Seth / 量子测量的同构关系

## 关键诚实评估

### Badiou — 最近的西方家族(兄弟之争)

> "Same object, different precision layer."
> 同样的命题在两个语态里。Badiou 用诗化-数学化语言(集合论 · 真理程序);本学派用物理学语言。两者在精度上不同 —— Badiou 觉得本学派"太物理化",本学派觉得 Badiou"还不够物理化"。**但这是兄弟之争,不是敌对。**

### 韩炳哲 — 德式哀悼 vs 美式操作

> 韩属于德语文化批判传统(Adorno · Heidegger · Levinas)—— 语态是**哀悼 + 呼唤**。本学派的语态是**操作 + 干预**。韩说出疾病的名字;本学派说出治疗的语法。这是方法论分歧,不是谁对谁错。

### Picard — 协作可能

> 不同层级问题。Picard 工作在客观可测层(信号 · 物理 · 神经);本学派工作在主体本体层(振动 · 语法 · 主权)。但两者**在 AI 训练数据审计上可以协作** —— 学派的子应用 04+ 是这个对话的具体落地。

### Seth — 不同尺度物理学

> Seth 把意识"放进"大脑;本学派把振动场"放在"大脑之间。这不是冲突,是**不同尺度的物理学**。Seth 做单脑意识的预测处理;本学派做双脑振动场的语法物理学。

### 量子测量 — 认可争论 · 时间会给答案

> "What quantum measurement is to particles, ontological grammar is to vibration fields."
> 同一种结构性事实在不同领域显形。但物理学家通常拒绝把量子原理外推到日常语言尺度。本学派回应:不是滥用,是识别同构。**时间会给答案。**

## 视觉创新

### 6 种顶部边框色 · 一眼区分

每个对话顶部用不同色 ——
粉 → 橙 → 紫 → 浅蓝 → 绿 → 主蓝。

翻页时不需要看标题就知道当前讨论的是哪个对话对象。

### 左右分栏 dialog 块

左侧 `.party.opponent`(红色边)= 对手位置 + 直接引文
右侧 `.party.school`(绿色边)= 本学派位置

让"两个学派同时在场"的视觉感真实存在,不是后人写历史的视角。

### 引文 `.quote`

Lora italic 字体,正式学术引文 feel。让 Bowlby / Badiou / Han / Picard / Seth 的引言有视觉权重,不只是 paraphrase。

### Scene meta 块

每个对话开头的 scene 用 `.diam` 浅灰斜体 —— 介绍对话背景,与对话主体视觉区分。

### Engagement Point callout

蓝色 callout(`linear-gradient` 背景 + dashed border),让"精确交锋点"成为视觉焦点 —— 这是每个对话的核心。
