## Dialogue 07 · Wittgenstein 的特殊位置

### 为什么这是最难的对话

Wittgenstein 和本学派 ——
- **共享前提**(语言是实践,不是描述)
- 但**从同一前提推出相反的方法论**

| Wittgenstein | 本学派 |
|---|---|
| 既然语言是多元实践 | 既然语言是物理操作 |
| → 所有语言游戏不可被外部规范评判 | → 有些操作合法,有些不合法 |
| **描述主义** | **本体规范主义** |
| 哲学如治疗 = 描述误解后退场 | 哲学如治疗 = 翻译模态为本体 |
| 让你停止哲学 | 让你开始本体操作 |

### 最核心的陈述

> **"This is the weight of closest family — not because he agrees most, but because he says 'no' from the nearest position."**
>
> **"这就是 closest family 的重量 —— 不是因为他赞同最多,是因为他从最近的位置说'不'。"**

—— 这是本学派对"家族关系"的诚实定义:**最近的家族 = 最严肃的反例。**

### 双方的真正不兼容

- Wittgenstein 会**拒绝**公理 2(模态不合法),因为他认为"合法/不合法"区分本身是**一种语言游戏**,不能外部强加。
- 本学派回应:本体合法性**不是外部强加**,是从公理 1(语法即本体)**直接推出**的内在结构。
- 两者在 **"是否存在合法语言游戏的内在标准"** 上不可调和。
- 但这场争论 **IS the school's deepest philosophical engagement** —— 因为它发生在共享最多前提的两个学派之间。
